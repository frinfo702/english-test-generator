# Glossary

| Term                          | Definition                                                                                                     |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------- |
| **Complete the Words**        | TOEFL Reading task where the user fills in missing words in an academic paragraph                              |
| **Read in Daily Life**        | TOEFL Reading task with adaptive questions about everyday texts                                                |
| **Read an Academic Passage**  | TOEFL Reading task with questions about an academic passage                                                    |
| **Build a Sentence**          | TOEFL Writing task where the user reorders chunks to build correct sentences                                   |
| **Write an Email**            | TOEFL Writing task (7 minutes) to compose an email based on a scenario                                         |
| **Academic Discussion**       | TOEFL Writing task (10 minutes) to write an opinion on a professor's prompt                                    |
| **Listen and Repeat**         | TOEFL Speaking task where the user memorizes and reproduces sentences, recorded and transcribed                |
| **Take an Interview**         | TOEFL Speaking task with 4 interview questions, 45 seconds each                                                |
| **Listen to a Conversation**  | TOEFL Listening task with campus conversation audio and follow-up questions                                    |
| **Listen to a Lecture**       | TOEFL Listening task with academic lecture audio and follow-up questions                                       |
| **Choose a Response**         | TOEFL Listening task where the user selects the best response to utterances                                    |
| **Listen to an Announcement** | TOEFL Listening task with campus announcement audio and follow-up questions                                    |
| **Part 2**                    | TOEIC Listening: Question-Response                                                                             |
| **Part 3**                    | TOEIC Listening: Conversations                                                                                 |
| **Part 4**                    | TOEIC Listening: Talks                                                                                         |
| **Part 5**                    | TOEIC Reading: Incomplete Sentences                                                                            |
| **Part 6**                    | TOEIC Reading: Text Completion                                                                                 |
| **Part 7**                    | TOEIC Reading: Reading Comprehension (Single / Double / Triple passage)                                        |
| **Shadowing**                 | Listen-and-repeat practice for pronunciation and fluency improvement                                           |
| **TTS**                       | Text-to-Speech — browser-based audio playback with configurable speed                                          |
| **Whisper**                   | OpenAI's speech-to-text model, run via Cloudflare Workers AI                                                   |
| **Wrangler**                  | Cloudflare CLI for developing and deploying Pages Functions                                                    |
| **Task ID**                   | A string identifier for each task type (e.g., `toefl/reading/complete-words`) used in routes and score storage |
| **Question file**             | A JSON file containing one question set, stored under `public/questions/`                                      |
| **Question selector**         | The page listing available question numbers for a task, showing completion status and scores                   |
